Two different PPHs — kept separate. The Planned Adjusted Building PPH (your corporate number, 118–124) is the headline reported up the chain: it spans all direct + indirect hour buckets, so it drives the building manifold only — total paid hours = total volume ÷ adj-PPH, cost/piece = 60 ÷ adj-PPH. The per-belt outbound LOAD PPH (~170–270, each belt's own canonical rate) is a separate operational rate that drives outbound load staffing: loaders = belt volume ÷ belt PPH ÷ paid-day (your boss's Column M). Leave "Outbound load PPH" blank to use each belt's historical rate, or set one to model a push. The adjusted PPH does not set belt staffing. Hit Print for the wall sheet.
